# NeuroState Analyzer

Десктопное приложение для анализа ЭЭГ-сигнала с визуализацией мозговых ритмов и оценкой состояния.

## Возможности

- Загрузка одноканального ЭЭГ в формате CSV
- Отображение временного ряда с плеером (Play / Pause / Stop)
- Полосовые фильтры: общий (0.5–30 Гц), Alpha (8–13), Beta (13–30)
- Спектральный анализ: PSD по методу Welch, мощность 5 ритмов
- Круговая диаграмма распределения ритмов
- Индексы состояния: расслабление, концентрация, сонливость
- Автоматическое текстовое заключение

## Требования

- Python 3.10+

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Запускать из папки `NeuroStateAnalyzer/`.

## Формат входного файла

CSV с разделителем `;`, без заголовка:

```
897.66558838;2.39215684
897.67846680;2.37254906
...
```

Две колонки: **время (сек)** и **амплитуда (мкВ)**.

## Использование

1. Откройте вкладку **Сигнал** → нажмите **Загрузить файл** → выберите CSV
2. Просмотрите сигнал, при необходимости примените фильтр
3. Управляйте воспроизведением кнопками Play / Pause / Stop
4. Перейдите на вкладку **Анализ** → нажмите **Запустить анализ**
5. Изучите PSD-график, диаграмму ритмов, карточки состояния и заключение
